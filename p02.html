ส่วนประกอบ visual studio

ส่วนประกอบของ Visual studio จะมี Google chromeและ XAMPP เพื่อที่จะทำการทดสอบและรันผล

Google Chrome

เจ้า Google Chrome ถือว่ามีจุดเด่นในการรักษาประสิทธิภาพด้านความปลอดภัยและปกป้องความส่วนตัวของผู้ใช้งานอย่างสูงสุดอีกทั้งสิ่งสำคัญที่ขาดไปไม่ได้คือความเร็วในการเปิดเว็บไซต์ต่างๆที่แตกต่างจาก Browser ทั่วไปอย่างสูงสุดอีกทั้งยังมีหน้าตาของโปรแกรมที่ใช้งานง่ายสะอาดสะอ้านและยังสามารถปรับแถบ taskbar และการคีย์ address Bar ต่างๆได้อย่างง่ายดายอีกทั้งยังมีการแสดงรายชื่อมัลแวร์คือซอฟต์แวร์ที่ประสงค์ร้ายต่อเครื่องของคุณและเตือนคุณทุกเมื่อที่มีการบุกรุก อีกทั้งยังมีความสามารถในด้าน V8 Javascript Engine ประมวลผล Javascript ได้รวดเร็ว

XAMPP

เจ้า XAMPP เป็นโปรแกรม Apache web server ไว้จำลอง web server เพื่อไว้ทดสอบ สคริปหรือเว็บไซต์ในเครื่องของเรา โดยที่ไม่ต้องเชื่อมต่ออินเตอร์เน็ตและไม่ต้องมีค่าใช้จ่ายใดๆ ง่ายต่อการติดตั้งและใช้งานโปรแกรม XAMPP จะมาพร้อมกับ PHPภาษาสำหรับพัฒนาเว็บแอพลิเคชั่นที่เป็นที่นิยม, MySQL ฐานข้อมูล, Apache จะทำหน้าที่เป็นเว็บ เซิร์ฟเวอร์, Perl อีกทั้งยังมาพร้อมกับOpenSSL ,phpMyadmin (ระบบบริหารฐานข้อมูลที่พัฒนาโดย PHP เพื่อใช้เชื่อมต่อไปยงัฐานขอ้ มูล สนับสนุนฐานข้อมูล MySQL และ SQLite โปรแกรม XAMPP จะอยู่ในรูปแบบของไฟล์Zip, tar, 7z หรือ exe โปรแกรม XAMPP อยู่ภายใต้ใบอนุญาตของ GNU General Public License แต่บางครั้งอาจจะมีการเปลี่ยนแปลงเรื่องของลิขสิทธิ์ในการใช้งาน
